
# ICS3UV

## Typescript with Deno

- to compile your program
  ```console
  deno run main.ts
  ```

- to format your program
  ```console
  deno fmt ./*.ts
  ```
- to lint your program
  ```console
  deno lint ./*.ts
  ```

## Go

- to compile your program
  ```console
  go run main.go
  ```
- to format your program
  ```console
  go fmt ./*.go
  ```
- to lint your program
  ```console
  golangci-lint run
  ```
